# gold-miner-web

[![Build Status](https://travis-ci.org/sqrthree/gold-miner-web.svg?branch=develop)](https://travis-ci.org/sqrthree/gold-miner-web)

掘金翻译计划 web 端，正在开发阶段。

## Table

- BackEnd: backend service.
- FrontEnd: Front-End service.

## Develop

```
cd FrontEnd
npm install
npm run dev
npm run mock
```

## Deploy

```
cd FrontEnd
npm install
npm run build
```
